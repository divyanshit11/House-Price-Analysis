# House Price Prediction

A machine learning project to predict house prices using structured housing data.

## What it does

* Cleans data and handles missing values
* Encodes categorical features using One-Hot Encoding
* Splits data into training and validation sets
* Trains a regression model (SVR)
* Evaluates performance using MAPE

## Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn

## Conclusion

* Data preprocessing and encoding are crucial for model performance
* Feature scaling is important for SVR
* Simpler models like Random Forest may perform better on tabular data
